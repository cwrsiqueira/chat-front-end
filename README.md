RODAR O PROJETO BAIXADO

Instalar todas as dependências 
### npm install

Rodar o projeto
### npm start 


CRIAR O PROJETO

Criar projeto React - https://reactjs.org/docs/create-a-new-react-app.html
### npx create-react-app my-app

Acessar o diretório do projeto
### cd my-app

Rodar o projeto
### npm start
